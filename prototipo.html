Protótipo

Em nosso projeto, criamos um protótipo usando os softwares AutoCAD e SolidWorks. O AutoCAD foi usado para esboços detalhados, enquanto o SolidWorks permitiu a criação de modelos 3D complexos. Além disso, desenvolvemos o aplicativo na plataforma Figma, utilizando o Visual Studio Code (VSCode) para escrever os códigos em HTML, CSS e JavaScript. A colaboração entre as equipes de design e desenvolvimento foi fundamental para o sucesso do projeto.
